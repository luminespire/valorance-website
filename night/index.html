Night

A thumb alpha keyboard layout.

Design

Night is an efficiency focused layout with specific targets for comfort. It differs from traditional layouts in using the letter R on the left thumb.

Efficient

Night features extremely low same finger and skip same finger usageEx. MY | MAY.

These enable Night to be efficient at high typing speeds.

Rhythmic

Night is an alternation focused layout.

This makes typing feel consistent and rhythmicEx. LAKE.

Fluid

Night focuses on reducing redirects, especially weak redirectsEx. SAD.

This minimizes awkward sequences.

Ergonomic

Night takes advantage of the characteristics of each finger.

This distributes typing load.
